class ModuleT [T] extends Module

A class to declare type parameterized module that can be consumed safely.

Due to type erasure it's techincally impossible to differentiate module which are constructed with parameterized types in a given context.

As a workaround this class allow to safely declare type parameterized module by using an implicit scala.reflect.Manifest.

final class StoreModule[T <: AnyRef](implicit manifest: Manifest[T], context: Context) extends ModuleT[T] {
  override val bindings: Bindings =
    bind[Store[T]] to new DefaultStore[T]

  def store = inject[Store[T]]
}
