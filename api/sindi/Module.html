Module

trait Module extends Context with Childable

A module is a collection of services which are configured using bindings.

The bindings are overridden at runtime using parent's Context.

It's recommended to:
- Declare class as final and prefer Module composition and Component mixing over class inheritance.
- Define an implicit Context provided as class constructor to ease contextual integration.

final class UserModule(override val ctx: Context) extends Module {
  override lazy val modules = new StoreModule[User](this) :: Nil

  override val bindings: Bindings =
    bind[UserService] to autowire[DefaultUserService]

  def users = inject[UserService]
}
